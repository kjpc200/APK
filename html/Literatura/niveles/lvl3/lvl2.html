Nivel 2 - Literatura

Las fichas de indexación son fichas utilizadas para resumir información relevante de un texto, conferencia, clase, o cualquier otra fuente de información.
Las fichas de estudio o resumen son fichas utilizadas en sistemas de indexación y catalogación para organizar y clasificar información de manera sistemática.
Las fichas bibliográficas son fichas utilizadas para registrar información bibliográfica de libros, revistas, artículos, etc., incluyendo datos como el título, autor, editorial, año de publicación, entre otros.
